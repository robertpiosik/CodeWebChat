# API tools

CWC includes battle-tested must-have API tools.

### [Code completions](./code-completions)

Get accurate code-at-cursor from state-of-the-art reasoning models.

### [Context edit](./context-edit)

Modify, create or delete files based on natural language instructions.

### [Intelligent update](./intelligent-update)

When applying chat response, update files based on code blocks in truncated edit format and fix malformed diffs.

### [Commit messages](./commit-messages)

Generate meaningful summaries of changes precisely adhering to your preferred style.
