CWC integrates context selection with the native file explorer making the process familiar and convenient.

## XML structure

```
<files>
  <text title="...">...</text>
  // other websites...
  <file path="...">...</file>
  // other files...
</files>
```

## Available commands

##### `Code Web Chat: Copy Context`

Copy context selection to clipboard.
